## Interaction Features Now Working
- **Mouse wheel:** zoom in/out centred on cursor
- **Mouse drag:** pan the camera
- **Touch pinch:** zoom in/out
- **Touch two-finger drag:** pan
- **Double-tap canvas:** reset view
- **Reset View button:** reset zoom and pan to default fit

## Animation Features Now Working
- **Play/Pause:** toggles RAF time advancement
- **Reverse:** flips orbital direction
- **Speed:** 0.25× to 365× days per second
- **Step buttons:** auto-pause, then jump ±1 or ±7 days
- **Date display:** robust for large offsets; pulses blue when playing
- **Background-tab safety:** dt capped to prevent time jumps

## Visual Features Now Working
- Procedural seeded starfield background (regenerable/copyable/pasteable)
- Faint nebula clouds behind stars (violet/indigo/blue/pink, 3–6 per viewport)
- Orbit rings for all INTRAS Level 1 bodies
- **Disk point fields:** 300–800 scattered points per circumstellar disk, warmer browns, radial jitter
- Filled-circle bodies coloured by type
- Main world highlighted with gold stroke + "MAIN" label
- Label culling at low zoom to reduce clutter
- Deterministic initial angles derived from system hash

## Performance Optimisations
- Single `requestAnimationFrame` loop (no per-object timers)
- Off-screen culling for non-disk bodies
- Label culling when zoom < 0.35×
- No DOM manipulation during animation; pure Canvas 2D
- PWA offline caching via MWG service worker
